## 🚀 Instalação

```bash
# Clone o repositório e entre na pasta do projeto
$ git clone https://github.com/RayanneRamos/react-notesweb.git
$ cd react-notesweb

# Instale as dependências
$ npm install
# ou
$ yarn install

# Execute a aplicação
$ npm start
# ou
$ yarn start
```

Abra http://localhost:3000 no seu navegador para ver a aplicação rodando!
